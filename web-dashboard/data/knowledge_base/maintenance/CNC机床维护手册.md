# CNC 数控机床维护手册

> 适用设备：100台 CNC 数控机床（CNC_001 ~ CNC_100）  
> 版本：v1.0  
> 编制：工业智能运维团队

---

## 一、日常点检标准

### 1.1 每日点检（开班前 30 分钟）

| 项目 | 检查内容 | 标准 | 异常处理 |
|------|---------|------|---------|
| 主轴运转 | 空转运行 5 分钟，监听异响 | 无异常噪音，转速稳定 | 停机检查轴承，联系 senior_technician |
| 电压检测 | 测量控制柜输入电压 | 380V ± 5%（三相） | 电压超出范围立即停机，通知 electrical_specialist |
| 电流检测 | 测量主轴电机工作电流 | 不超过铭牌额定值 110% | 降低进给速度，检查负载 |
| 温度巡检 | 红外测温枪检测主轴轴承座 | 温升 ≤ 35°C/h，绝对温度 ≤ 80°C | 加强冷却，检查润滑系统 |
| 冷却液 | 检查液位和浓度 | 液位 ≥ 60%，浓度 3-5% | 补充或更换冷却液 |
| 急停按钮 | 按下测试，复位 | 按下后主轴立即停止 | 更换急停开关 |
| 防护门联锁 | 开门测试 | 门开后主轴自动停止 | 维修或更换联锁开关 |

### 1.2 每周点检（每周一开班前）

| 项目 | 检查内容 | 标准 |
|------|---------|------|
| 润滑系统 | 检查导轨油和主轴油油位 | 油位在标尺 1/2 以上 |
| 过滤器 | 清洁或更换液压站空气滤清器 | 无明显堵塞 |
| 排屑器 | 清理切屑，检查传送带 | 运转正常，无卡滞 |
| 电气柜 | 检查柜内温度、清洁度 | 温度 ≤ 45°C，无积尘 |
| 刀具夹紧力 | 测试刀具夹紧/松开 | 夹紧力 ≥ 额定值 90% |
| 编码器电缆 | 外观检查，接头紧固 | 无破损，无松动 |

### 1.3 每月点检（每月第一个工作日）

| 项目 | 检查内容 | 标准 |
|------|---------|------|
| 主轴精度 | 千分表测量主轴跳动 | 径向跳动 ≤ 0.005mm |
| 导轨间隙 | 塞尺测量各轴导轨间隙 | ≤ 0.02mm |
| 反向间隙 | 激光干涉仪测量 | X/Y ≤ 0.01mm, Z ≤ 0.015mm |
| 水平度 | 水平仪检查机床水平 | ≤ 0.02mm/m |
| 电气接线 | 全面紧固控制柜接线端子 | 扭矩达标，无氧化 |
| 接地电阻 | 测量保护接地 | ≤ 4Ω |
| 液压站 | 检查液压油品质，测试压力 | 油质清澈，压力稳定 |

## 二、主轴系统维护

### 2.1 轴承更换步骤

**适用轴承型号**：7014C/P4（角接触陶瓷球轴承）

**所需工具**：
- 轴承加热器（感应式，80-100°C）
- 扭矩扳手（0-100 Nm）
- 千分表及磁力表座
- 专用拆卸拉马（液压式）
- 无尘布、无水酒精
- 高温润滑脂（Kluber NBU 15 或等效）

**更换步骤**：

1. **断电挂牌**：执行 LOTO（Lockout-Tagout）程序，确认主电源断开
2. **拆卸主轴箱盖**：用内六角扳手卸下主轴箱上盖螺栓，取下上盖
3. **拆除旧轴承**：
   - 拆下锁紧螺母和止动垫圈
   - 用液压拉马将轴承从主轴上拉出
   - 检查主轴轴颈有无划伤或磨损
4. **清洁**：用无水酒精清洁主轴轴颈和轴承座孔，确保无杂质
5. **安装新轴承**：
   - 将新轴承放入轴承加热器，加热至 90°C
   - 戴隔热手套，快速将轴承推入主轴轴颈
   - 等待冷却至室温后，安装锁紧螺母
6. **调整预紧力**：用扭矩扳手以 25 Nm 拧紧锁紧螺母，然后回松 1/8 圈
7. **测量跳动**：用千分表测量主轴前端径向跳动，应 ≤ 0.003mm
8. **试运转**：
   - 低速（500 RPM）运行 10 分钟，监听异响
   - 逐步升速至额定转速（8000 RPM），每 2000 RPM 运行 5 分钟
   - 监测轴承温升，正常 ≤ 25°C/h

**注意事项**：
- 轴承不可直接敲击，必须使用加热法或专用工具安装
- 安装时保持环境清洁，防止灰尘进入
- 新轴承运行前 8 小时为磨合期，期间降低负载 30%

### 2.2 主轴润滑

- **润滑方式**：油脂润滑（高速主轴）或油雾润滑
- **润滑周期**：每 2000 运行小时或 3 个月（以先到为准）
- **加脂量**：轴承空间 30-40%，过量会导致温升
- **推荐油脂**：Kluber NBU 15, SKF LGHP 2, 或 Shell Gadus S2 V100

### 2.3 主轴对中

1. 用千分表检查主轴与刀柄锥面的接触率 ≥ 80%
2. 主轴拉刀力测试：用拉力计测量，应 ≥ 额定值 90%
3. 主轴与工作台垂直度调整：用直角尺 + 千分表，≤ 0.01mm/300mm

## 三、电气系统维护

### 3.1 电压检测标准

| 参数 | 正常范围 | 告警阈值 | 停机阈值 |
|------|---------|---------|---------|
| 输入电压 | 380V ± 5% | ±8% | ±10% |
| 控制电压 | 24V DC ± 5% | ±8% | ±10% |
| 主轴驱动 DC 母线 | 540V DC ± 5% | ±10% | ±15% |

### 3.2 接线检查

1. **每月**：紧固所有接线端子，包括主电源、控制电源、信号线
2. **每季度**：热成像扫描电气柜，排查过热点
3. **每半年**：绝缘电阻测试（500V 兆欧表），各回路对地 ≥ 1MΩ

### 3.3 常见电气故障

| 故障现象 | 可能原因 | 排查方法 |
|---------|---------|---------|
| 主轴无法启动 | 急停未复位、变频器故障、电机过载 | 检查急停回路 → 变频器报警代码 → 电机绝缘 |
| 驱动器报警 | 过流、过压、编码器信号丢失 | 查看驱动器面板故障码，断电复位 |
| 控制面板无显示 | 电源模块故障、24V 电源异常 | 测量 24V 输出，更换电源模块 |

## 四、冷却系统维护

### 4.1 散热器清洗

1. **周期**：每 500 运行小时或每月
2. **方法**：用压缩空气（≤ 4 bar）从内向外吹扫散热器翅片
3. **深度清洗**：每季度用中性清洗剂喷淋后清水冲洗，晾干后装回
4. **检查**：翅片无倒伏、无堵塞，风扇运转无异响

### 4.2 导热硅脂更换

1. **周期**：每年或更换功率模块时
2. **步骤**：
   - 拆下功率模块
   - 用无水酒精清除旧硅脂
   - 均匀涂抹新硅脂（厚度约 0.1mm）
   - 重新安装模块，对角线均匀紧固螺栓

### 4.3 冷却液管理

- **浓度检测**：每周用折光仪检测，切削液浓度 3-5%
- **pH 值**：8.5-9.5，低于 8.0 需更换
- **更换周期**：每 3 个月或出现异味、浑浊时

## 六、维护记录规范

每台设备维护后应记录：
1. 设备编号和时间
2. 维护类型（日常点检 / 计划维护 / 故障维修）
3. 更换的备件名称和数量
4. 维修前/后 Z-Score 对比
5. 验收结果（PASS / FAIL / INCONCLUSIVE）
6. 技师签名和日期

记录通过 AI Copilot 的工单跟踪系统自动归档，可在 `/work-order-tracking` 页面查询完整历史。
